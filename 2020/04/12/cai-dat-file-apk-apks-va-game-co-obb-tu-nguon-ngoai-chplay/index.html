Cài đặt file Apk, Apks và Game có OBB từ nguồn ngoài CHplay

Những điều cần biết

Tiềm hiểu 1 tẹo về các file APK, APKS và OBB. Các file này sử dụng để cài đặt các ứng dụng, trò chơi trên điện thoại hoặc các thiết bị sử dụng hệ điều hành Android. Và thường sử dụng khi cài đặt từ các nguồn không chính thống từ Play Store (CHplay).

Tệp APK

APK là một tệp từ khi tôi sử dụng điện thoại Android đã có :D Nó được sự dụng để cài phần mềm đã được chữa lại như những game, ứng dụng tính phí.
Trích lại 1 nguồn về file APK như sau: APK là viết tắt của cụm từ tiếng Anh “Android application package”. APK là một định dạng file nén. Nó tương tự như ZIP, dựa trên định dạng JAR, và sử dụng phần mở rộng là .apk.*
Đơn giản như trên PC thì có file *.exe tải về mở ra cài đặt là có game chơi, phần mềm để dùng chứ không phải là copy cái icon hay shortcut rồi về mở ra để nghiên cứu nhé.

Tệp APKS

APKS thì cũng là 1 file dùng để cài đặt như APK, như tôi hiểu thì công dụng của nó giống như file APK. Tuy nhiên lằng nhằng hơn 1 điểm là không cài đặt như APK chỉ cần ấn vào là cài. File APKS khi cài lại cần 1 phần mềm nữa, tôi hiện đang dùng SAI (Split APKs Installer) :D

Còn cụ thể APKS là gì tôi trích lược 1 đoạn như sau: APKs là một định dạng mới được sử dụng gần đây. Với sự xuất hiện của Android App Bundle, việc phát hành và tối ưu hóa các ứng dụng trên Play Store trở nên dễ dàng hơn rất nhiều đối với các nhà phát triển. Tuy nhiên, nó khiến người dùng không thể xuất và chia sẻ các tệp APK như bình thường nữa. Thay vào đó, bạn cần sử dụng tới APKs. Về bản chất, một tệp APKs là tập hợp của nhiều file APK khác nhau. Ví dụ, nếu bạn muốn cài đặt ứng dụng Swiftkey bản Mod, tệp tin bạn tải về sẽ gồm 4 file APK nhỏ và cho dù bạn cài đặt tệp APK nào thì cũng nhận được một thông báo lỗi.

Tệp OBB

OBB là phần dữ liệu bổ sung của trò chơi hoặc ứng dụng, nó chứa các bộ sưu tập các tập tin khác nhau như: âm thanh, đồ họa, kịch bản … Một số game bắt buộc có OBB để hoạt động, một số thì không. Thông thường, các tệp OBB có kích thước lớn và cần một công cụ phù hợp để giải nén. Nếu bạn thường xuyên tải và cài đặt các trò chơi có dung lượng cao, bạn sẽ gặp OBB file thường xuyên. Thường là các file game hoặc phầm mềm đã được thuốc.

Cách sử dụng OBB file thì rất đơn giản. Bạn không cần cài đặt nó như tệp APK hay APKs, chỉ cần di chuyển nó tới đúng thư mục yêu cầu để đảm bảo trò chơi có thể hoạt động ổn định.

Hương dẫn cài đặt file APK, APKS và OBB

Cài File APK

Đơn giản nhất trong tất cả, chỉ cần tải file về và ấn cài đặt. Với các file cài đặt không từ nguồn của CHPlay, khi nhấp vào cài đặt sẽ cần xác nhận Nguồn không xác định Hoặc bật trong Cài đặt => Bảo Mật => Nguồn không xác định
Với các phiên bản Android mới sẽ cung cấp quyền cho các ứng dụng lẻ, ví dụ như bạn tải ứng dụng về từ Chrome thì khi hoàn tất tải về và nhấp cài đặt hệ thống sẽ hỏi cấp quyền cài đặt ứng dụng từ nguồn là Chrome.

Cài File APKS

Sau khi cài đặt ứng dụng SAI (Split APKs Installer)

  1. Mở ứng dụng SAI
  2. Ấn vào mục Install APKs
  3. Chọn thư mục có file APKs cần cài đặt
  4. Chọn ứng dụng cần cài đặt => Ấn vào select => Ứng dụng sẽ bắt đầu mở file, các bạn cần chờ 1 chút sau đó bảng cài đặt tương tự cài đặt file APK sẽ hiện lên và bắt đầu cài đặt tương tự file APK

Cài File OBB

Thường các file OBB là 1 thư mục dữ liệu và được nén lại dưới định dạng .zip hoặc .rar.
Vì vậy bạn cần giải nén file này ra bạn nhận được một thư mục thường có tên dạng com.developer-name.androidpackage.
Tiếp theo, di chuyển nó tới đường dẫn đích SDCard: Android/OBB

Sau khi đã hoàn tất thì mở ứng dụng hoặc trò chơi lên và thưởng thức thôi.
